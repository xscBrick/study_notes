前几天，苹果悄悄上架了2020版iPad Pro，延续了前代11寸和12.9寸的大小设计，并加入了激光雷达扫描仪的后置双摄模组，支持Wi-Fi6。但新机型才上线不久，国外就有知情人士爆料称，苹果年底将推出5G版iPad Pro，搭载A14X处理器。


但5G版iPhone 12还未发布，下一代iPad却跃跃欲试，这似乎是在暗示今年的iPhone 12或许也将搭载5G网络。


据消息称，年底将推出的这版iPad Pro采用的A14X处理器，基于台积电5nm工艺进行打造。5G技术方面，将外挂骁龙X55基带。屏幕方面则将采用mini-LED背光的LCD显示面板，在对比度、亮度等方面表现更好。


不过相信大家最关注的还是价格，相对于6229元起售的2020版iPad Pro来说，5G版的iPad Pro或许会更贵一些，加上众多用户会选择购买Apple Pencil或者键盘来说，近10000元的价格可以购买一台不错的电脑了。
